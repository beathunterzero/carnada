ENTORNO DE DESARROLLO PYTHON - REFERENCIA ESTÁNDAR

Versión de Trabajo: Python 3.12
Sistema Operativo: Ubuntu (WSL)
Última revisión: Abril 2026
Próxima revisión: Abril 2027

================================================================

1. INSTALACIÓN BASE DEL SISTEMA (ejecutar una sola vez)

sudo apt update
sudo apt install python3 python3-pip
sudo apt install python3.12-venv python3.12-full


2. CREACIÓN DEL ENTORNO VIRTUAL (MÉTODO RECOMENDADO)

# Crear el entorno virtual
python3.12 -m venv .venv

# Activar el entorno virtual
source .venv/bin/activate

# Actualizar pip
python -m pip install --upgrade pip


3. CREACIÓN ALTERNATIVA (en caso de error)

# Eliminar entorno anterior si existe
rm -rf .venv

# Crear entorno sin pip
python3.12 -m venv .venv --without-pip

# Activar
source .venv/bin/activate

# Instalar pip manualmente
python3 -m ensurepip --upgrade

# Actualizar pip
python -m pip install --upgrade pip


4. VERIFICACIÓN

python --version     # Debe mostrar Python 3.12.x
pip --version


================================================================

REGLAS OBLIGATORIAS PARA TODOS LOS PROYECTOS PYTHON

- Siempre usar "python3.12 -m venv .venv" para crear el entorno virtual.
- Nunca usar solo "python3 -m venv .venv".
- Trabajar siempre dentro del entorno virtual (debe verse (.venv) en el prompt).
- Dentro del venv, se puede usar "python" o "python3" indistintamente.
- Actualizar pip cada vez que se crea un nuevo entorno.
